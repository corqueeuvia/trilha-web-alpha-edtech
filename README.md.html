Mas... o que são tags?

Tags HTML

São marcadores pra dar nome aos bois em uma página.
Os bois, no caso, são os pedaços de texto e outros elementos que você coloca na página (imagens, vídeos, botões, tudo!).

Agora que você já se apaixonou por código, vamos listar umas tags e falar sobre elas:

  • <h1> marca um título. Vai até <h6>, diminuindo de tamanho
  • <p> marca um parágrafo
  • <head> marca a "cabeça" da página, onde a pessoa dev vai colocar um monte de tralha que não vem ao caso
  • <body> marca o lindo corpinho dessa página sensual (e de todas as outras não tão sensuais assim) e é o que você está olhando agora, então sem safadeza!

Atributos

Eis que do nada falamos em atributos, dizendo que <meta> foi usada com um atributo.
Também falamos que <html> é o elemento raiz do rolê todo.
Então é isso: o nome técnico-real-oficial do que estamos chamando de tags é "elemento".
Os elementos podem ter atributos, que são uma parte extra pra colocar informações que você não quer mostrar na página, mas necessárias para sua formatação.
(tipo selecionar o conjunto de caracteres do padrão UTF-8 pra acentuação rodar direito na página)

Curiosidade:

Como atributos do elemento <img> (que ainda não mencionamos), nessa página foram passados os valores:

  • src="imagens/print-codigo-pagina-html.png" que indica a origem da imagem, onde a página deve buscá-la pra exibir aqui.
  • alt="Print do código desta página HTML" que indica uma alternativa, caso a imagem não carregue, a ser exibida pra você.
Ou seja, pra colocar a imagem ali em cima, fiz assim:
<img src="imagens/print-codigo-pagina-html.png" alt="Print do código desta página HTML">
Outra coisa que ainda não te contei é sobre o elemento <title>
Ele é colocado dentro de <head> e é uma das "tralhas" que pessoas desenvolvedoras usam.
É o que está marcado como title que ficará no topo da página, nomeando a aba desta página aberta no seu navegador.
